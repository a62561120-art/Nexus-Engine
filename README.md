Nexus Engine Pro
​Controls

​Move: WASD / Arrows (PC) or Joystick (Mobile).
​Jump: Space (PC) or Jump Button (Mobile).
​Look: Click + Drag (PC) or Swipe (Mobile).
​Select: Click or Tap an object.

​Quick Guide
​Terrain: Pick a sculpt mode and click/tap the ground to make hills or holes.
​Assets: Paste a direct link to any .glb or image to change models/textures.
​Sound: Upload an MP3. Use Spatial for 3D sound or Touch for triggers.
​Logic: Write JS in the script box. It runs once you export.

​Exporting
​Hit Export APK. It gives you an HTML file. Send that file to anyone—it runs in any browser on any device.
